核心功能

三层记忆架构
全面覆盖你的需求

🧬

长期记忆（LTM）

持久化存储你的个人档案、偏好、习惯和目标。每次对话开始时自动加载，AI 无需你重新介绍自己。支持按分类管理，敏感信息可加密存储。


支持分类：profile · preference · habit · goal · experience

preference 喜欢简洁代码风格，偏好中文注释
goal 目标是在 6 个月内完成马拉松训练
habit 每天早上 6 点起床，跑步 5 公里
profile 后端工程师，主要使用 Python / FastAPI
📚

个人知识库（KB）

将技术笔记、参考资料、解决方案结构化存储。对话中随时调用，AI 回答时自动参考你的私有知识，不依赖公共训练数据。

⚡

会话记忆（STM）

追踪当前任务上下文，记录会话事件和待保存内容。会话结束时统一整理，决定哪些内容值得写入长期记忆。

💡

智能触发引擎

实时分析对话内容，自动识别偏好表达、目标陈述、习惯描述等 13 类可记忆模式，主动提示保存，无需手动判断。

🔍

BM25 语义搜索

基于 BM25Okapi 算法的全文检索，支持中英文混合分词。在数千条记忆中精准定位相关内容，毫秒级响应。

工作原理

四步启动协议
每次任务自动执行

新建任何任务时，系统在后台自动完成以下四步，无需手动干预。

1

Session Start

创建会话实例，记录任务类型，开始追踪本次对话上下文

2

Memory Profile

加载用户档案，让 AI 了解你的背景、偏好和身份信息

3

Memory Recall

根据当前任务主题，检索并注入相关的历史记忆条目

4

KB Search

从个人知识库中查找相关文档和笔记，作为参考上下文
