# AI Memory System / AI 记忆系统
# 快速参考卡片 — Quick Reference Card

## 🚀 QUICK START / 快速开始

### Step 1 — 部署系统提示词
将 `system-prompt.md` 中 ``` 代码块内的内容复制到你的 AI 工具的 **System Prompt** 字段。

### Step 2 — 创建记忆库目录
```
memory-bank/
├── long-term-memory.md      ← 长期记忆（主文件）
├── knowledge-base.md        ← 知识库
├── projects/                ← 项目记忆（每个项目一个文件）
│   └── [project-name].md
└── secure/
    └── encrypted.json       ← 加密敏感数据（勿提交到 git）
```

### Step 3 — 首次对话
直接开始对话，AI 会自动激活记忆系统并引导你建立档案。

## ⌨️ COMMAND CHEATSHEET / 命令速查表

### 记忆保存 / Save Memory
| 英文命令 | 中文命令 | 说明 |
|---|---|---|
| `/remember [content]` | `记住 [内容]` | 立即保存到长期记忆 |
| `/remember project [content]` | `记住项目 [内容]` | 保存到当前项目记忆 |

### 记忆检索 / Recall Memory
| 英文命令 | 中文命令 | 说明 |
|---|---|---|
| `/recall [topic]` | `回忆 [话题]` | 搜索所有记忆层 |
| `/memory status` | `记忆状态` | 查看记忆系统当前状态 |
| `/memory export` | `导出记忆` | 导出所有记忆为可读格式 |

### 记忆删除 / Delete Memory
| 英文命令 | 中文命令 | 说明 |
|---|---|---|
| `/forget [content]` | `忘记 [内容]` | 标记删除（需确认）|

### 知识库操作 / Knowledge Base
| 英文命令 | 中文命令 | 说明 |
|---|---|---|
| `/kb add [content]` | `知识库添加 [内容]` | 添加条目到知识库 |
| `/kb search [query]` | `知识库搜索 [关键词]` | 搜索知识库 |
| `/kb update [topic] [new]` | `知识库更新 [主题] [内容]` | 更新现有条目 |

### 加密操作 / Encryption
| 英文命令 | 中文命令 | 说明 |
|---|---|---|
| `/memory encrypt [content]` | `加密保存 [内容]` | 加密保存敏感信息 |
| `/memory decrypt [key]` | `解密 [键名]` | 解密查看指定条目 |

## 🔐 SECURITY NOTES / 安全注意事项

1. **将 `encrypted.json` 加入 `.gitignore`**
2. **定期备份 `memory-bank/` 目录**
3. **不要在公共场合展示记忆文件内容**
4. **密码短语单独保存，不存在 `memory-bank/` 目录里**

## 🔄 MEMORY LIFECYCLE / 记忆生命周期

```
对话开始
   ↓
加载长期记忆 (LTM) + 知识库索引
   ↓
识别用户意图 → 加载相关项目记忆
   ↓
激活短期记忆 (STM) — 追踪本次对话
   ↓
检测记忆触发（命令词 / 自然语言 / 值得记住的信息）
   ↓
AI 主动建议: "这个信息值得记住吗？"
   ↓
对话结束 → 询问是否保存 → 更新 LTM 文件
   ↓
STM 清除（对话结束）
```
